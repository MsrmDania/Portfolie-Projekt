HTML

HTML er strukturen på et website, og bestemmer hvordan den rå version af hjemmesiden ser ud. HTML inkluderer overskrifter, brødtekster, billeder og links til andre sider. HTML arbejder i Tags, som hjælper søgemaskiner med at vise relevant indhold og bygger strukturen op inde i HTML’en. Tags består af Meta Tags og Semantiske Tags.

Tags

  • Meta Tags er tags som har en usynlig effekt på HTML, men fortællere serveren og browseren
    essentiel information, så som sprog på siden og forbindelse til CSS.
  • Semantiske Tags er vigtig, og bliver anvendt til søgemaskine optimering. Semantiske tags
    fortæller søgemaskinen, hvilke sektioner af websitet den skal fremvise i preview, og finder
    relevant information i teksten.

F.eks. Det semantiske tag, section er en af de mest udbredte semantiske tags. Tagget fungerer som en division, eller div, deler ikke kun indeholder op og fortæller søgemaskinen at brødtekst eller indhold starter og slutter inden for tagget.

Et eksempel:

<html lang="da">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1">
<link href="../css/bootstrap.min.css" rel="stylesheet">
<link href="../css/portfolie.css" rel="stylesheet">
</head>

Billeder og Links

Med HTML er det muligt at indsætte billeder og links direkte ind i websitens rå kode.

Billeder

For at sætte et billede ind i et HTML dokument anvender man et img tag. Et eksempel:

<img src=”billede” alt=”billede tekst”>

I tilfælde af at browseren af en eller anden grund ikke kan loade billedet in på websiten anvender siden alternativ tekst, eller ”alt”, hvor man kan udfylde en lille beskrivelse af billedet. Alt tekst bruges også til skærmlæsere.

”src” fortæller browseren kilden på billedet, altså dens lokation inde i web-serveren, eller hvis den er hentet online.

Links

Links er essentielle for at navigere rundt mellem websites, men kan også anvendes som intern navigation på din egen side. For at lave et navigationslink til din skal du anvende et a href link, f.eks.:

<a href=”http://www.hjemmeside.dk”>Tekst som kan klikkes</a>

Ved at angive en klikbar tekst, erstattet et linkets udseende med teksten.

A indikerer at det er et link.

Href fortæller browseren hvor linket fører hen.

Class og ID

Class og ID er anvendt til at skabe funktioner for CSS med, Class bliver anvendt til elementer på siden, som gentager sig flere gange igennem hele sidens funktion, mens ID beskriver ét specifikt element og hvilke CSS ændringer den skal have.

Et Eksempel:

<div class="en_class"></div>
<div id="et_ID"></div>

Indhold skrevet inden for Class og ID tags bliver påvirket af deres CSS varianter.
